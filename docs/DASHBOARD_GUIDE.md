## 1. Πρόσβαση στο Dashboard

```
URL: http://localhost:3000/admin/
```

> Σε production: `https://api.yourdomain.gr/admin/`

---

## 2. Εγγραφή Εργοδότη

1. Ανοίξτε `http://localhost:3000/admin/`
2. Πατήστε **"Εγγραφή"**
3. Συμπληρώστε:
   - **Επωνυμία Εταιρείας**: π.χ. "ΚΕΝΤΡΙΚΗ ΚΑΦΕΤΕΡΙΑ ΑΕ"
   - **ΑΦΜ**: 9 ψηφία (π.χ. `123456789`)
   - **Email**: info@company.gr
   - **Κωδικός**: τουλάχιστον 8 χαρακτήρες
4. Πατήστε **Εγγραφή** → αυτόματη σύνδεση

> ⚠️ Για sandbox/testing χρησιμοποιήστε δοκιμαστικό ΑΦΜ `000000000`

---

## 3. Σύνδεση

- Email + Κωδικός → **Σύνδεση**
- Το session διαρκεί 8 ώρες
- Μετά λήξη, αυτόματο redirect στο login

## 5. Διαχείριση Εργαζομένων

### Προσθήκη

1. Σελίδα **Εργαζόμενοι** → **"+ Νέος Εργαζόμενος"**
2. Συμπληρώστε:
   - **Επώνυμο**: ΚΕΦΑΛΑΙΑ (π.χ. `ΠΑΠΑΔΟΠΟΥΛΟΣ`)
   - **Όνομα**: ΚΕΦΑΛΑΙΑ (π.χ. `ΓΕΩΡΓΙΟΣ`)
   - **ΑΦΜ**: 9 ψηφία
   - **Παράρτημα**: Επιλέξτε
   - **Εξωτερικός**: ☑️ αν bypass geofence
3. Αυτόματα δημιουργείται **linking code**

> ⚠️ Επώνυμο/Όνομα ΠΡΕΠΕΙ να ταιριάζουν ΑΚΡΙΒΩΣ με ΕΡΓΑΝΗ!

### Αναζήτηση

Γράψτε στο πεδίο αναζήτησης — φιλτράρει live.

### Απενεργοποίηση

Soft delete — ο εργαζόμενος δεν μπορεί να κάνει check-in.

## 6. Κωδικοί Σύνδεσης

### Τι είναι;

Ο linking code είναι **6ψήφιος κωδικός** που ο εργαζόμενος πληκτρολογεί στο chatbot για να συνδεθεί.

### Δημιουργία

1. Εργαζόμενοι → πατήστε **🔗 Κωδικός** στον εργαζόμενο
2. Εμφανίζεται ο κωδικός (π.χ. `549237`)
3. Δώστε τον στον εργαζόμενο (SMS, email, ή αυτοπροσώπως)
4. Ο κωδικός λήγει αυτόματα σε 7 ημέρες

### Τι κάνει ο εργαζόμενος

```
1. Ανοίγει Telegram/Viber/WhatsApp
2. Βρίσκει το bot "Κάρτα Εργασίας"
3. Πληκτρολογεί: 549237
4. ✅ "Καλωσόρισες, ΓΕΩΡΓΙΟΣ ΠΑΠΑΔΟΠΟΥΛΟΣ!"
5. Στέλνει τοποθεσία → check-in!
```

## 7. Μαζική Εισαγωγή CSV

### Βήματα

1. Εργαζόμενοι → **📄 CSV Import**
2. Εισάγετε JSON array:

```json
[
  {"afm": "111111111", "eponymo": "ΠΑΠΑΔΟΠΟΥΛΟΣ", "onoma": "ΓΕΩΡΓΙΟΣ"},
  {"afm": "222222222", "eponymo": "ΔΗΜΗΤΡΙΟΥ", "onoma": "ΜΑΡΙΑ", "phone": "+306912345678"},
  {"afm": "333333333", "eponymo": "ΚΩΝΣΤΑΝΤΙΝΟΥ", "onoma": "ΝΙΚΟΣ"}
]
```

3. Πατήστε **📥 Εισαγωγή**
4. Κάθε εργαζόμενος λαμβάνει αυτόματα linking code

### Αποτέλεσμα

Εμφανίζεται: `X επιτυχίες`, `Y αποτυχίες` + λεπτομέρειες αποτυχιών

## 8. Διαχείριση Παραρτημάτων

### Προσθήκη

1. Παραρτήματα → **"+ Νέο Παράρτημα"**
2. Συμπληρώστε:
   - **Όνομα**: π.χ. "Κεντρικό Κατάστημα"
   - **Αριθμός ΕΡΓΑΝΗ**: `0` = κεντρικό
   - **GPS**: Latitude + Longitude (από Google Maps)
   - **Geofence**: 10–200 μέτρα (default: 40m)

### GPS: Πώς βρίσκω;

1. Google Maps → δεξί κλικ στο κατάστημα
2. Αντιγράψτε: `37.97550, 23.73480`
3. Latitude = `37.97550`, Longitude = `23.73480`

## 9. Ιστορικό Βαρδιών

### Φίλτρα

- **Ημερομηνία** (από → έως)
- Export CSV

### Πίνακας δείχνει:

| Στήλη       | Περιγραφή                          |
| ----------- | ---------------------------------- |
| Εργαζόμενος | Ονοματεπώνυμο                      |
| Ενέργεια    | ✅ Είσοδος / 🚪 Έξοδος               |
| Ημ/νία Ώρα  | Χρονοσήμανση                       |
| Παράρτημα   | Πού ήταν                           |
| Geofence    | approved/rejected/bypassed         |
| ΕΡΓΑΝΗ      | pending/submitted/confirmed/failed |

### Export CSV

Πατήστε **📥 Export CSV** → κατεβαίνει αρχείο `.csv` με UTF-8 encoding.

## 10. Fraud Alerts

Εμφανίζει ύποπτες δραστηριότητες:

| Τύπος Alert         | Σημασία                                      |
| ------------------- | -------------------------------------------- |
| `gps_spoofing`      | GPS coordinates ίδιες ακριβώς (Fake GPS app) |
| `impossible_travel` | Μετακίνηση πολύ γρήγορα μεταξύ σημείων       |
| `zero_accuracy`     | GPS accuracy = 0 (ύποπτο)                    |

### Εξέταση

Πατήστε **✅ Εξετάστηκε** → σημειώνεται ημερομηνία + ποιος.

## 12. API Reference

Όλα τα endpoints του Admin API:

### Authentication

| Method | URL                        | Περιγραφή                      |
| ------ | -------------------------- | ------------------------------ |
| POST   | `/api/admin/auth/login`    | Σύνδεση (email+password → JWT) |
| POST   | `/api/admin/auth/register` | Εγγραφή νέου εργοδότη          |
| GET    | `/api/admin/auth/me`       | Τρέχων χρήστης                 |

### Dashboard

| Method | URL                          | Περιγραφή                    |
| ------ | ---------------------------- | ---------------------------- |
| GET    | `/api/admin/dashboard/stats` | Σημερινά στατιστικά          |
| GET    | `/api/admin/dashboard/chart` | Δεδομένα γραφήματος 7 ημερών |

### Εργαζόμενοι

| Method | URL                                     | Περιγραφή                              |
| ------ | --------------------------------------- | -------------------------------------- |
| GET    | `/api/admin/employees`                  | Λίστα (?search=, ?branchId=, ?active=) |
| POST   | `/api/admin/employees`                  | Προσθήκη                               |
| PUT    | `/api/admin/employees/:id`              | Ενημέρωση                              |
| DELETE | `/api/admin/employees/:id`              | Απενεργοποίηση                         |
| POST   | `/api/admin/employees/:id/linking-code` | Νέος κωδικός                           |
| POST   | `/api/admin/employees/import-csv`       | Μαζική εισαγωγή                        |

### Παραρτήματα

| Method | URL                       | Περιγραφή |
| ------ | ------------------------- | --------- |
| GET    | `/api/admin/branches`     | Λίστα     |
| POST   | `/api/admin/branches`     | Προσθήκη  |
| PUT    | `/api/admin/branches/:id` | Ενημέρωση |

### Χρονοσημάνσεις

| Method | URL                            | Περιγραφή                                                       |
| ------ | ------------------------------ | --------------------------------------------------------------- |
| GET    | `/api/admin/timestamps`        | Ιστορικό (?dateFrom=, ?dateTo=, ?employeeId=, ?status=, ?page=) |
| GET    | `/api/admin/timestamps/export` | Export CSV                                                      |

### Fraud Alerts

| Method | URL                                  | Περιγραφή          |
| ------ | ------------------------------------ | ------------------ |
| GET    | `/api/admin/fraud-alerts`            | Λίστα (?reviewed=) |
| PUT    | `/api/admin/fraud-alerts/:id/review` | Σημείωση εξέτασης  |

### Ρυθμίσεις

| Method | URL                   | Περιγραφή           |
| ------ | --------------------- | ------------------- |
| GET    | `/api/admin/settings` | Τρέχουσες ρυθμίσεις |
| PUT    | `/api/admin/settings` | Ενημέρωση           |

### Authentication Headers

```
Authorization: Bearer <JWT_TOKEN>
Content-Type: application/json
```

### Παράδειγμα curl

```bash
# Login
TOKEN=$(curl -s -X POST http://localhost:3000/api/admin/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"12345678"}' | jq -r '.token')

# Dashboard stats
curl -H "Authorization: Bearer $TOKEN" http://localhost:3000/api/admin/dashboard/stats

# Λίστα εργαζομένων
curl -H "Authorization: Bearer $TOKEN" http://localhost:3000/api/admin/employees

# Νέος εργαζόμενος
curl -X POST -H "Authorization: Bearer $TOKEN" -H "Content-Type: application/json" \
  http://localhost:3000/api/admin/employees \
  -d '{"afm":"987654321","eponymo":"ΤΕΣΤ","onoma":"ΧΡΗΣΤΗΣ"}'
```
